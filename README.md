# 2024-ITCS371-2-Group8

Travel Naja is a new travel booking platform that aims to make trip planning easy and
complete for users. Our client wants to stand out in the busy travel booking market by
offering a strong system that can compete with well-known platforms like Booking.com and
Agoda. Travel Naja offers three main services: flight bookings, car rentals, and hotel
reservations. It also has a special feature that lets users book local tour guides, adding a
personal touch to their trips.
